**************************************************
	PSP File Browser Version 1.00
**************************************************
PSPのファイルブラウザです。
綺麗なintraFontを使ってファイルブラウザを作って見ました。
PSP Filerを意識して作っていますがツリー型表示には対応していません。
これから機能を増やしていければと思っています。

また、EBOOT.PBPと同じ階層にBackground.pngという名前の画像を配置すると
背景として表示することができます。

１.操作方法
↑ボタン：1つ上のフォルダ・ファイルを選択
↓ボタン：1つ下のフォルダ・ファイルを選択
○ボタン：選択しているフォルダを開く
	：.txt/.db/.iniファイルならテキストビューアーで表示
	：.pngファイルならイメージビューアーで表示
	：.mp3ファイルならミュージックプレイヤーで再生
	：対応していないファイルならどのプログラムで開くか選択。
×ボタン：開いているファイルを閉じる
△ボタン：メインメニュー
□ボタン：ファイルメニュー
Rボタン	：次ページ
Lボタン	：前ページ
